# movie-app-nextjs

## Prerequirement
- Node: 18.17.1

## Migrate tables
`npx prisma migrate dev --name init`

## Start the server
- `npm run dev`

## Api Doc
- Api doc available on `/api-doc` url

## Deployment
- `npm run build`
- `npm run start`
